# Анализ дампа памяти и сети

## Установка

1. Установите Python 3.7 или выше.
2. Установите необходимые библиотеки, используя команду:
   pip install pyshark matplotlib seaborn
   
   3. Установите Volatility, следуя инструкциям на официальном сайте.

## Запуск анализа

1. Скачайте файл `dhcp.pcapng` и поместите его в папку с проектом.
2. Запустите скрипт:
     python analysis.py
